# 🔗 Biolink - David Alonso

Um biolink moderno e responsivo que funciona como página de perfil pessoal e profissional. Desenvolvido com HTML5, CSS3 e um design elegante em modo escuro.

## 🎯 Sobre

Este projeto é um **Biolink** (Biografia + Link) - uma página única que centraliza todos os links importantes e informações profissionais de **David Alonso Nogueira Silva**, estudante de Desenvolvimento de Sistemas com foco em backend.

## ✨ Características

- ✅ Design moderno com tema escuro
- ✅ Totalmente responsivo (mobile-first)
- ✅ Gradiente azul com efeitos visuais
- ✅ Animações suaves
- ✅ Links para projetos e redes sociais
- ✅ Otimizado para performance

## 🎨 Customização

### Mudar a Foto de Perfil
Substitua o arquivo de imagem em `assets/` e atualize o caminho em `index.html`:
```html
<img src="assets/sua-foto.jpg" alt="Seu Nome" id="avatar">
```

### Editar Informações Pessoais
Modifique o conteúdo do arquivo `index.html`:
- Nome na tag `<h1>`
- Descrição na tag `<p>`
- Links das redes sociais e projetos nos `href`

### Alterar Cores
Edite as variáveis CSS no arquivo `style.css`:
```css
:root {
    --accent: #4a9eff;  /* Cor de destaque */
    --primary-gradient: linear-gradient(...);  /* Gradiente de fundo */
}
```
